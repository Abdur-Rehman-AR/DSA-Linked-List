DSA Linked List in Java

This repository contains Java implementations of Linked Lists and related algorithms. It covers Singly Linked List, Doubly Linked List, Circular Linked List, and common operations like Insertion, Deletion, Traversal, Searching, Sorting, and Advanced Manipulations.

All programs are written from scratch to help understand how nodes connect, how data moves, and how dynamic memory works in Linked Lists without relying on Java built-in classes.
